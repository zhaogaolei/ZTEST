## 安装

```bash
# 通过 npm 安装
npm i zmjd-mobile -S
```

## 快速上手

```js
import Vue from 'vue';
import { Button } from 'zmjd-mobile';
Vue.use(Button);
```

## Project setup
```
npm install
```

### Compiles Demo and hot-reloads for development
```
npm start
```

### Compiles Demo and minifies for production
```
npm run build
```

### 构建组件-打包
```
npm run build-c
```
